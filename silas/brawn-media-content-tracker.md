## CONTENT PRODUCTION CHECKLIST (Per Page)

### Pre-Writing

- [ ] Content brief created (keyword, audience, goals, outline)
- [ ] Competitor research completed (top 5 ranking pages analyzed)
- [ ] Internal linking targets identified (hub + 3-5 related pages)
- [ ] Visual requirements documented (images, diagrams, screenshots)
- [ ] CTA strategy defined (primary, secondary, tertiary)

### Writing

- [ ] Outline approved by strategist
- [ ] Draft completed at target word count
- [ ] Primary keyword in title, first paragraph, headers
- [ ] 5+ internal links added with descriptive anchors
- [ ] 2-3 external authority links added (cite sources)
- [ ] FAQ section added (5-8 questions)
- [ ] CTA sections added (top, middle, bottom of page)

### Post-Writing

- [ ] Editor review completed
- [ ] SEO check (Yoast/RankMath or similar)
- [ ] Images added with alt text and captions
- [ ] Schema markup added (Service, FAQ, LocalBusiness)
- [ ] Meta title and description written (under character limits)
- [ ] Internal links from existing pages added
- [ ] Page published and submitted to Google Search Console

### Post-Launch (Week 1)

- [ ] Traffic/impressions check in GSC
- [ ] Keyword ranking check
- [ ] Conversion tracking verified
- [ ] Social media promotion (LinkedIn, Facebook)
- [ ] Newsletter mention (if applicable)
