3 SIMPLE STEPS TO DEPLOY
================================================================================

STEP 1: INSTALL PLUGIN (5 minutes)
────────────────────────────────

1. Go to WordPress Dashboard
2. Click: Plugins → Add New
3. Search: "Code Snippets"
4. Click: INSTALL NOW
5. Click: ACTIVATE

STEP 2: ADD SCHEMA CODE (10 minutes)
────────────────────────────────

1. In Dashboard: Click Snippets → Add New
2. Copy ALL code from file: chicago_electrician_schema_deploy.php
3. Paste into code area
4. Set: "Run everywhere"
5. Click: Save Changes and Activate Snippet

STEP 3: VERIFY & TEST (15 minutes)
────────────────────────────────

A) Verify in Source Code:
   1. Go to: https://www.chicagoselectrician.com
   2. Press: Ctrl+U (View Source)
   3. Search: "application/ld+json"
   4. You should see: <script type="application/ld+json">

B) Validate with Google:
   1. Go to: https://search.google.com/test/rich-results
   2. Enter: https://www.chicagoselectrician.com
   3. Click: TEST URL
   4. You should see: ✓ No errors
