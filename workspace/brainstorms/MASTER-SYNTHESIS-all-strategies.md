### Theme 5: CONVERSION OPTIMIZATION
**Strategy:** #13 (Customer Magnet Framework)

**Core insight:** Most SEO agencies get traffic but don't convert. The Customer Magnet Framework balances 3 content types to move readers from awareness â†’ trust â†’ transaction.

**The 3 content types:**
1. **Traffic content** - Gets eyeballs (how-to guides, listicles, beginner topics)
2. **Trust content** - Builds authority (case studies, strategy breakdowns, proof)
3. **Transaction content** - Converts buyers (comparisons, reviews, "best X" articles)

**Common mistake:**
> "Most people only create traffic content. They get 10,000 visitors but no customers."

**The fix:**
- 70% traffic + trust content (build authority, attract visitors)
- 30% transaction content (convert high-intent traffic)

**LocalCatalyst application:**
- Audit existing content by type (categorize each article)
- Identify gaps (probably need more transaction content)
- Create comparison articles ("LocalCatalyst vs DIY SEO")
- Create buyer-intent articles ("Best Local SEO for Restaurants")
