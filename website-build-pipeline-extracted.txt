Why a Dedicated Design Agent
Builder assembles WordPress pages from templates and content. But someone needs to decide what those templates look like â€” color palette, typography, layout structure, spacing, visual hierarchy, image treatment. That's design, and it's a different skillset from CMS assembly.

Canvas bridges the gap between "we have the client's logo and brand colors" and "here's a fully designed page layout Builder can implement."
Canvas â€” Agent Profile
AGENT: Canvas

ROLE: Visual design system creation and art direction

WHAT CANVAS DOES:

  1. Takes client brand assets (logo, colors, existing materials)

  2. Generates page layout specifications that Builder follows

  3. Selects and curates stock photography / generates image assets

  4. Creates visual mockups for client review

  5. Defines responsive behavior (how layouts adapt to mobile)

WHAT CANVAS DOES NOT DO:

  - Write content (that's Scribe)

  - Build WordPress pages (that's Builder)
